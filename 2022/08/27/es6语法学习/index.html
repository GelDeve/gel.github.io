ECMASript 6

1、let 关键字

let 关键字用来声明变量，以后声明变量使用 let 就对了

（1）不允许重复声明

（2）块儿级作用域

(全局, 函数, eval)

eval 通常用来执行一个字符串表达式，并返回表达式的值

（3）不存在变量提升

（4）不影响作用域链

在提到块儿级作用域时，会认为在块儿级作用域里面才有用，如果说里面声明函数的话，也不可以使用，是不对的

fn()里面没有，向上一级找，就找到了school

所以说虽然是块级作用域，但是不会影响作用域链的效果

（5）小案例

var没有块级作用域，所以它一直是在全局中存在的，循环完成后window.i的值就是3了，那么，items[3]越界了，找不到目标元素，所以无法修改

解决办法：将var改成let

2、const 关键字

const 关键字用来声明常量

（1）声明一定要赋初始值

const A; // 报错

（2）一般常量使用大写(潜规则)

// const a = 100;

（3）常量的值不能修改

（4）块儿级作用域

（5）对于数组和对象的元素修改, 不算做对常量的修改, 不会报错

声明对象类型使用 const，非对象类型声明选择 let

从数组和对象中提取值，对变量进行赋值，这被称为解构赋值。

频繁使用对象方法、数组元素，就可以使用解构赋值形式

6、箭头函数

ES6 允许使用「箭头」（=>）定义函数。

箭头函数不会更改 this 指向，用来指定回调函数会非常合适

（1）this 是静态的. this 始终指向函数声明时所在作用域下的 this 的值

（2）不能作为构造实例化对象

（3）不能使用 arguments 变量

arguments是一个对应于传递给函数的参数的类数组对象。

有了这个对象我们以后写函数的时候，就不用给所有的形参指定参数名，然后通过参数名的方式获取参数了，我们可以直接使用arguments对象来获取实参

（4）箭头函数的简写

  1. 省略小括号，当形参有且只有一个的时候
  1. 省略花括号，当代码体只有一条语句的时候，此时 return 必须省略

而且语句的执行结果就是函数的返回值

filter()把传入的函数依次作用于每个元素，然后根据返回值是true还是false决定保留还是丢弃该元素。

箭头函数适合与 this 无关的回调. 定时器, 数组的方法回调

箭头函数不适合与 this 有关的回调. 事件回调, 对象的方法

10、Symbol

ES6 引入了一种新的原始数据类型 Symbol，表示独一无二的值。它是JS语言的第七种数据类型，是一种类似于字符串的数据类型。

遇到唯一性的场景时要想到 Symbol

（1）Symbol的特点

1）Symbol 的值是唯一的，用来解决命名冲突的问题

2）Symbol()与ymbol.for() 方法

使用Symbol.for() 方法创建时，会将创建的Symbol存入一个全局的注册表中，在下次创建时，会首先从注册表中查找是否已有相同的值存在，如果没有，则创建；如果有，便不会创建重复的Symbol。

所以使用Symbol.for() 创建的两个Symbol判断是否相等时，返回的时true。

3）Symbol 值不能与其他数据进行运算

4）USONB => （你太牛逼了）

  • U：undefined
  • S：string、symbol
  • O：object
  • N：null、number
  • B：boolean

（2）Symbol 创建对象属性

为了防止你创建的方法和对象本身的方法和冲突，选择在另一个对象中使用Symbol创建独一无二的方法，用[ ]调用，不会污染本身对象中的方法

为对象添加独一无二的属性，不能直接使用Symbol ()，因为Symbol是一个动态值，不是固定属性，所以将Symbol放在[ ]中，即[Symbol(‘say’)]

（3）Symbol 内置值

11、迭代器

遍历器（Iterator）就是一种机制。它是一种接口，为各种不同的数据结构提
供统一的访问机制。任何数据结构只要部署 Iterator 接口，就可以完成遍历操作。

（1）ES6 创造了一种新的遍历命令 for…of 循环，Iterator 接口主要供 for…of 消费

（2）可用 for of 遍历：

Array、Arguments、Set、Map、String、TypedArray、NodeList

（3）for…in 和 for…of 区别：

for in遍历的是数组的索引（即键名），而for of遍历的是数组元素值。

（4） 工作原理：

a) 创建一个指针对象，指向当前数据结构的起始位置
b) 第一次调用对象的 next 方法，指针自动指向数据结构的第一个成员
c) 接下来不断调用 next 方法，指针一直往后移动，直到指向最后一个成员
d) 每调用 next 方法返回一个包含 value 和 done 属性的对象

iterator 接口就是对象的一个属性，即Symbol.iterator 属性

（5）迭代器应用

需要自定义遍历数据的时候，要想到迭代器。

13、promise

Promise 是 ES6 引入的异步编程的新解决方案。语法上 Promise 是一个构造函数，
用来封装异步操作并可以获取其成功或失败的结果。

（1）基本语法

1）要想创建一个promise对象，可以使用new来调用Promise的构造器来进行实例化。

2）Promise 构造函数包含一个参数和一个带有 resolve（解析）和 reject（拒绝）两个参数的回调。在回调中执行一些操作（例如异步），如果一切都正常，则调用 resolve，否则调用 reject。

3）当异步代码执行成功时，我们才会调用resolve(…), 当异步代码失败时就会调用reject

4）使用setTimeout(…)来模拟异步代码，实际编码时可能是XHR请求或是HTML5的一些API方法.

（2）promise封装读取文件

14、Set

ES6 提供了新的数据结构 Set。它类似于数组，但是成员的值都是唯一的，没有重复的值。

set集合实现了 iterator 接口，所以可以使用『扩展运算符』和『for…of…』进行遍历，

  4. has 检测集合中是否包含某个元素，返回 boolean 值

15、map

ES6 提供了 Map 数据结构。它类似于对象，也是键值对的集合。但是“键”的范围不限于字符串，各种类型的值（包括对象）都可以当作键。

Map 也实现了iterator 接口，所以可以使用『扩展运算符』和『for…of…』进行遍历。

Map 的属性和方法：

  1. size 返回 Map 的元素个数
  2. set 增加一个新元素，返回当前 Map
  3. get 返回键名对象的键值
  4. has 检测 Map 中是否包含某个元素，返回 boolean 值
  5. clear 清空集合，返回 undefined
